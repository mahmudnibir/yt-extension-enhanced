Features

Built for the way
you actually watch.

Every control you've wanted but had to live without — speed, audio DSP, filters, bookmarks, shortcuts. All in one free extension.

Precision Speed Control

Slide from 0.25× to 20× with a smooth dial. Fine-step with +/− keyboard shortcuts on any page globally.

Universal Hover Overlay

Hover any <video> element on any website. A compact control bar floats over it instantly.

Auto Ad Skip

YouTube ads are detected and skipped the moment they load. Zero clicks required.

Cloud Bookmarks

Timestamp any moment with a label. Synced across all your Chrome sessions via the storage API.

5 Voice Modes

Real-time Web Audio DSP — Chipmunk, Bass Boost, Robot, Echo, or Normal. Works at any speed.

Content Filters

Hide feed videos by keyword, channel, or duration. Keep recommendations intentional.

Watch Statistics

Track total watch time and session history in a live graph broken down by day and week.

Custom Keyboard Shortcuts

Every control has a fully remappable hotkey. Open the shortcuts tab, click a binding, press your combo. Changes apply instantly without a page refresh.

Universal Overlay

Hover any video.
Get instant controls.

A minimal floating bar appears when your cursor enters any video element — YouTube, Twitter, Reddit, anywhere. No extra setup.

Web Audio DSP

Five ways to hear
your content.

Real-time audio processing applied directly on the media element via the Web Audio API. No quality loss, zero latency spikes.

Normal

Clean passthrough. Natural pitch locked at any speed.

Chipmunk

High-shelf boost at 2500 Hz. Cartoon-high, classic effect.

Bass Boost

Low-shelf gain at 200 Hz. Deeper, punchier audio.

Robot

Ring modulation oscillator. Synthetic, robotic texture.

Echo

Delay node with feedback. Spacious room reverb effect.

Installation

Running in
four steps.

Side-load directly from GitHub — no Chrome Web Store review wait required.

Step 01

Clone the repo

git clone https://github.com/mahmudnibir/yt-extension-enhanced

Step 02

Open Extensions

Go to chrome://extensions and enable Developer mode via the top-right toggle.

Step 03

Load Unpacked

Click Load unpacked and select the cloned project folder.

Step 04

Pin & use

Pin the icon to your Chrome toolbar. Open YouTube and click the icon to start.
